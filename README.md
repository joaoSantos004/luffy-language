## PL-luffy

This is an interpreted programming language themed with luffy.

## features

- Variable assignment
- User input
- Print with and without newline
- Addition
- Subtraction
- Multiplication
- Modulo
- String reversal
- If/else conditons
- Loops

## keywords and operators

- `free` : create or assign a variable
- `meat` : get user input
- `gomu` : print without newline
- `gomuno` : print with newline
- `gatling` : multiplication
- `crew` : addition
- `g3` : subtraction
- `nika` : modulo
- `ace` : reverse a string
- `straw` : if
- `hat` : else
- `gear` : loop

## how to run

remember this is not python but jython

```command prompt
examples:
jython interpreter.py helloworld.txt
jython interpreter.py multiply.txt
jython interpreter.py repeater.txt
jython interpreter.py reverse.txt
jython interpreter.py palindrome.txt
jython interpreter.py is_even.txt
```

## Syntax examples

### Variables

```txt
free x = 5
free name = 'Luffy'
```

### User input

```txt
meat user_input
```

### Printing

```txt
gomuno 'Hello'
gomu 'A'
gomu 'B'
gomuno ''
```

### Operations

```txt
free x = 5 crew 3 (result = 8)
free y = 10 g3 4 (result = 6)
free z = 6 gatling 7 (result = 42)
free r = 9 nika 2 (result = 1)
```

### Reversing strings

```txt
free backwards = word ace
```

### If/else statement

```txt
straw rem == 0 [
gomuno 'even'
]
hat [
gomuno 'odd'
]
```

### Loop

```txt
gear 5 [
gomu '*'
]
```
